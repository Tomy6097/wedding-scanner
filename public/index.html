Downloads a file with all phone numbers and individual invitation links. Use it to set up a WhatsApp Broadcast List.

How it works:
1. Click the button below — a .csv file downloads
2. Open the file in Excel or Google Sheets
3. Copy each guest's personal link
4. Send each guest their link via WhatsApp
5. Each guest opens their link to see their QR code
