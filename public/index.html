Bulk Import (CSV)

CSV format: name,phone,table_number,ticket_type

Column guide:
1. name — Full name (required)
2. phone — Phone number e.g. 255712345678 (optional)
3. table_number — Table or seat e.g. Table 5, VIP (optional)
4. ticket_type — S = Single (1 person), D = Double (2 persons). Default: S

Example:
name,phone,table_number,ticket_type
John Smith,255712345678,Table 1,S
Jane & Peter,255712345679,Table 2,D
